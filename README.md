🗺️ craft_layout

Geometric Optimization & Automated Cartography for ArcGIS Pro Map Series

Standard Map Series often suffer from "one-size-fits-all" orientations, resulting in wasted white space or inefficient scales for features with high geometric variability. craft_layout is a Python-driven toolkit that treats layout selection as a spatial optimisation problem—programmatically determining the best-fit orientation and scale for every feature before automating the high-resolution export process.

🚀 The Workflow
1. Geometric Analysis (suggest_layout.py)

This module performs a "best-fit" calculation by projecting each feature's bounding box against the usable area of your map frames.

Orientation Logic: Compares feature extents against the aspect ratios of Portrait and Landscape frames.

Scale Optimisation: Calculates the maximum possible scale for each orientation, incorporating a user-defined safety margin (default 5%).

Data Enrichment: Appends LAYOUT (P/L) and PAGE_SCALE fields to the attribute table, creating a data-driven foundation for the Map Series configuration.

2. Contextual Production (batch_export.py)

This module automates the export of finished PDFs while adding professional cartographic "focus."

Dynamic Switching: Automatically routes features to their optimised layout (P/L) during a single batch run.

Automated Context Layering: Generates a temporary "grey-out" layer for every page. It applies transparency and desaturation to surrounding features, ensuring the primary subject is prominent while maintaining essential geographic context.

Production Safeguards: Includes a "Test Mode" for rapid prototyping at lower resolutions before committing to full high-DPI production.

🛠️ Technical Highlights

Unit Agnostic: Dynamically converts between Millimeters, Centimeters, Inches, and Points based on Layout page settings.

CRS Validated: Ensures the input layer is in a Projected Coordinate System to maintain linear distance and scale accuracy.

Enterprise Ready: Robust handling of File Geodatabases, Shapefiles, and SDE environments via managed edit sessions.

Headless Execution: Designed to run via the ArcGIS Pro Python window or as a formal Script Tool (.atbx).

📖 Setup
Project Requirements

An ArcGIS Pro project containing two Layouts named:

A3_PORTRAIT

A3_LANDSCAPE

Each layout must have an active Map Series configured using the fields generated by the suggester script.

Usage

Analyse: Run suggest_layout.py on your target polygon layer to generate orientation and scale attributes.

Export: Run batch_export.py to generate the final PDF deliverables.
